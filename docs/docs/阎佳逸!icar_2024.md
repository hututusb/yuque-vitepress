#### 介绍
本项目为长安大学第十九届全国智能汽车大赛完全模型组上位机代码开源。使用了多线程架构，拆分任务，提升巡线速度 感谢沈阳航空航天大学TUP-Robomaster实验室，上海交通大学AuTop，湖北工业大学蓝电yyds以及sasu官方的开源
#### 安装教程

1. cd icar
2. mkdir build
3. cd build
4. cmake ..
5. make -j8
#### 使用说明

1. 在终端中，build文件夹下（已经构建过代码）使用如下指令启动程序
```
./icar
```
